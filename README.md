# WeMeeting Api

WeMetting api

## 启动项目

```bash
$ git clone git@github.com:dadaowuma/we-meeting-api.git

$ cd we-meeting-api && npm install

# 执行测试
$ npm test

# 启动客户端API项目
$ npm start # localhost:3000

# 启动API文档项目
$ npm run doc-api # localhost:3001

# 启动开发规范
$ npm run doc-dev # localhost:3002
```
